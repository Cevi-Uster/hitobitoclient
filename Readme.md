# Cevi.DB Client
## Einführung

Cevi.DB Client bietet Zugriff auf die Mitgliederdatenbank des Cevi Schweiz [](https://db.cevi.ch). Es ermöglicht konfigurierbare Exporte in CSV- und XLS-Dateien.
